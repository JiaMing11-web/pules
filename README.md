# Pulse

AI 原生兴趣流平台概念项目，使用 Next.js、React、TypeScript、Tailwind CSS 和 Framer Motion 构建。

## Run

```powershell
npm.cmd install
npm.cmd run dev
```

打开 `http://localhost:3000`。

## 预览修复

如果浏览器出现黑底裸 HTML、样式丢失，或 `127.0.0.1:3001` 无法访问，直接双击：

```text
start-pulse-preview.cmd
```

这个脚本会自动停止 3001 旧进程、清理开发缓存，并重新启动预览。

## 页面

- `/` AI 信息流首页
- `/states` 兴趣状态中心
- `/discovery` AI 探索模式
- `/creator` 创作者智能中心
- `/engine` AI 推荐引擎
